# expense-tracker-python
## Features
- Add expenses
- View expenses
- Total calculation
- Category-wise analysis

## Tech Stack
- Python
- CSV (File Handling)

## How to Run

1. Install Python
2. Run the file:
   python expense_tracker.py
3. Follow menu options
A menu-driven Expense Tracker built using Python, featuring CSV-based data storage, total expense calculation, and category-wise analytics.
